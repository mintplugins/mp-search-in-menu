== Description ==

Show a magnifying glass in your menu which turns into a search bar on roll-over. Purely CSS powered.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the ‘mp-search-in-menu’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
